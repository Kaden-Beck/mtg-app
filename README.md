# Project Overview

This is a full stack project aiming to resolve some headaches with managing my MTG collection.

I am developing this very much in the open to practice communicating technical decisions and the process of learning new technologies.

The front end will be using TanStack Start in React and TypeScript.

The API layers will be written with GraphQL.

The server functions and server functions will be using python with FastAPI, Strawberry, Celery, and SQLAlchemy.

The backend and hosting will be on my local server with PostgreSQL and Tailscale.

## Server Backend

Using Postgres-SQL and Redis in a Docker Compose Container on my local server (repurposed Framework Mainboard) to be accessed on my tailscale network
